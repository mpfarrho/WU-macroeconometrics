# Macroeconometrics (AT) / Advanced Macroeconometrics (ST)
This repository contains course materials and information for several lectures on macroeconometrics at WU Vienna. This includes "Macroeconometrics" ([detailed syllabus](https://github.com/mpfarrho/WU-macroeconometrics/blob/main/syllabus-AT.md)) from the Applied Track (AT) and "Advanced Macroeconometrics" ([detailed syllabus](https://github.com/mpfarrho/WU-macroeconometrics/blob/main/syllabus-ST-adv.md)) from the Science Track (ST).
